typescript环境

环境搭建

  • 初始化node项目
  • 安装typesript(强类型)
  • 安装typings(typings为typscript的类型定义)
npm init
npm install --save-dev typescript    
npm install --save-dev typings
// or npm   npm install --save-dev typings typings
tsc --init     //初始化 tsconfig.json
typings init     //初始化 typings.json

tsconfig.json

tsconfig.json为typescript的配置文件
tsc –init生成的配置如下：

{
"compilerOptions": {
    "module": "commonjs",
    "target": "es5",
    "noImplicitAny": false,
    "sourceMap": false
},
"exclude": [
    "node_modules"
    ]
}

typings.json文件（*.d.ts）

javascript中的this

this

javascript中的this指向执行环境，而非定义环境，如果没有明确的定义，则指向全局变量或者undefined

var someone = {
    name: "Bob",
    showName: function(){
        alert(this.name);
    }
};
var other = {
    name: "Tom",
    showName: someone.showName
}
other.showName();　　//Tom
this定义在someone中，但是通过other.showName()调用，执行环境是other,所以输出是Tom

var deck = {
suits: ["hearts", "spades", "clubs", "diamonds"],
cards: Array(52),
createCardPicker: function() {
        return function() {
            var pickedCard = Math.floor(Math.random() * 52);
            var pickedSuit = Math.floor(pickedCard / 13);

            return {suit: this.suits[pickedSuit], card: pickedCard % 13};
        }
    }
}
var cardPicker = deck.createCardPicker();
var pickedCard = cardPicker();

alert("card: " + pickedCard.card + " of " + pickedCard.suit);

基本数据类型

接口

接口的作用就是对值的类型进行契约检查

interface Shap{
    color:string;
    size:string;
}
class Circle implements Shap{
    color:string;
    size:string;
    radius:string
}
var circle:Circle=new Circle();
