typescript环境

  • typescript

环境搭建

  • 初始化node项目
  • 安装typesript(强类型)
  • 安装typings(typings为typscript的类型定义)
npm init
npm install --save-dev typescript    
npm install --save-dev typings
// or npm   npm install --save-dev typings typings
tsc --init     //初始化 tsconfig.json
typings init     //初始化 typings.json

tsconfig.json

tsconfig.json为typescript的配置文件
tsc –init生成的配置如下：

{
"compilerOptions": {
    "module": "commonjs",
    "target": "es5",
    "noImplicitAny": false,
    "sourceMap": false
},
"exclude": [
    "node_modules"
    ]
}

typings.json文件（*.d.ts）

安装typescript类型文件

typings install dt~jquery --save --global

angular2中typescript的文件

typings install dt~core-js --save --global
typings install dt~jasmine --save --global
typings install dt~node --save --global

webpack

webpack为打包工具
webpack.config.js为webpacke的配置文件
