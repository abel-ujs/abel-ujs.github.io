typescript语法基础

基本数据类型

基本数据类型：number string boolean array tuple enum any void

var tupleExam: [string,number];
tupleExam=['10',1] //正确
tupleExam=['10','10'] //错误

enum Color{Red=1,Green,Blue}
var c:Color = Color.Green

接口

接口的作用就是对值的类型进行契约检查

interface Shap{
    color:string;
    size:string;
}
class Circle implements Shap{
    color:string;
    size:string;
    radius:string
}
var circle:Circle=new Circle();
